变速比转向器沿齿条长度方向改变齿的压力角 α₂（及模数），使小齿轮节圆半径随齿条位置变化，从而使角传动比 iω 随转向盘转角 β 变化。

由式 (8-21)：ip = Dsw / (2a) · iω0
力传动比与角传动比成正比。合理设计 iω 变化规律，在轻便性与灵敏性之间取得最优平衡。

曲线 3 — 上凸形，中部 iω 最大（高速不过敏），两端 iω 减小（低速大转角灵敏）。
乘用车有助力转向，省力由助力装置解决；高速直行时，希望方向盘不要过度灵敏——轻微扰动不应过度响应，防止高速摆头失稳。

中部 iω 大 → ip 大 → 同样的路面力，手上感觉轻，且对转向盘小角度输入响应较迟钝，高速直行稳定；
两端 iω 小 → 低速大转角时转向响应快，泊车灵活。

教材规定中部值不宜低于 15～16（防止高速过于敏感）。
推荐范围：乘用车 iω = 17～25

循环球式转向器 · 变速比原理（对比齿扇节圆半径）
原理说明
循环球式转向器中，齿扇（扇形齿轮）各齿的节圆半径 r 沿摆动方向逐渐变化。

由式 (8-15)：iω = 2πr / P（P 为蜗杆螺距）
角传动比与齿扇当前啮合位置的节圆半径 r 成正比。

因此，改变齿扇各齿节圆半径的分布规律，即可实现角传动比随转向盘转角 β 的连续变化——这与齿轮齿条式通过改变压力角的原理完全对应。
中间齿扇节圆半径 r₀ 小
中部 iω 小 → 灵敏
两端 iω 大 → 省力
↔ 曲线 1，曲线 2 · 商用车
中间齿扇节圆半径 r₀ 大
中部 iω 大 → 省力不过敏
两端 iω 小 → 灵敏
↔ 曲线 3 · 乘用车
